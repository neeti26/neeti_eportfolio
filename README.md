# Neeti Malu — Portfolio

Personal e-portfolio built with React + Vite + Tailwind CSS.

## Tech Stack
- React 19
- Vite 8
- Tailwind CSS v4
- Lucide React (icons)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) → New Project
3. Import the GitHub repo
4. Framework preset: **Vite**
5. Root directory: leave as-is (or set to `neeti-portfolio` if in a monorepo)
6. Click **Deploy**

That's it — Vercel auto-detects Vite and handles everything.
